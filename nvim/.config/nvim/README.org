#+title: Configuración de Neovim

Este es el archivo para configurar Neovim. Puesto que no es mi editor principal,
y a penas lo estoy aprendiendo a usar, las configuraciones son bastante simples.
Así como son simples, son entendibles, por lo que tampoco tienen comentarios.

Para (re)generarlas ejecuta ~M-x org-babel-tangle~.

#+begin_src lua :tangle init.lua

vim.opt.number = true
vim.opt.relativenumber = true
vim.opt.cursorline = true
vim.opt.wrap = false
vim.opt.expandtab = false
vim.opt.tabstop = 4
vim.opt.shiftwidth = 4
vim.opt.clipboard = "unnamedplus"
vim.opt.swapfile = false

#+end_src
